# ShoppingMall

## 기능 소개
### 1. 업로드 페이지 ↑
```
1-1. 업로드 기능
    - react-dropzone을 이용하여 이미지 드래그 앤 드랍 구현
    - multer를 이용하여 이미지 저장
    - 이미지 클릭하여 지우기
```
    
### 2. 랜딩 페이지 🏠
```
2-1. 카테고리에 따른 분류
    - 체크 박스 필터링

2-2. 검색 기능

2-3. Pagination
    - 하단의 '더보기' 버튼을 이용한 paging 기능 구현
    - 8개 단위로 리스트업
    - 더이상 불러올 데이터가 없는 경우 버튼 보이지 않기
```
